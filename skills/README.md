# Skills 索引

本目录只放项目级 skills。不要默认全量读取。

## 使用方式

1. 先按 `.ai-rules/task-routing.md` 判断任务等级和 skill 路由。
2. 只读取当前任务命中的 0-2 个 `skills/*/SKILL.md`。
3. 命中后必须遵守 `.ai-rules/skill-contract.md`。

## 补充本地 skill

默认 project-first。项目 skill 不足时，才可读取用户本地 skill 补充，并先说明缺口和本地 skill 名称。

本地 skill 不能覆盖项目红线、任务等级、验证要求和交付格式。
